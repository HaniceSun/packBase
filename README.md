## Overview

packBase is a template repository for creating new Python packages.

## Installation

```
git clone git@github.com:HaniceSun/packBase.git
mv packBase new_package_name
cd new_package_name
python setup.py

#create a new repo named new_package_name on GitHub and push the code
```
