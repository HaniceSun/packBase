## Overview

packBase is a simple template repository for creating new Python packages.

## Installation

```
git clone git@github.com:HaniceSun/packBase.git
pip install .
```

## Usage

```
mkidir new_package
cd new_package
packBase

create a new repo named new_package on GitHub and push the code
```
